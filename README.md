# morAdmin

一个Maven项目，Java Web的上课练习。

界面显示基于layui开源前端模板weAdmin

[layui](https://www.layui.com/)

[weAdmin](https://lovetime.gitee.io/weadmin/)

# 简介

正在学习Java Web开发，把上课同步做的练习放到这，第一次尝试，代码有些粗糙，如果有感兴趣的欢迎一起学习。
